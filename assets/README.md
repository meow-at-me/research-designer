# assets/

Self-made or CC0 assets only (GUIDELINES §4). No marketplace, no unclear-license files.

```
assets/
  models/    3D models (.glb/.gltf), textures, HDRIs — self-made or CC0; ship a poster image per model
  lottie/    .lottie / .json motion files — self-made or CC0
  previews/  per-theme preview stills and short GIFs/MP4s of the live demos
  shots/     own logo files (dark-bg + light-bg variants) — the only image-in-demo exception
```

Notes:
- Motion/3D can't be shown in a static screenshot, so each theme should also have a short looping
  capture (GIF/MP4 ≤ a few seconds) in `previews/` for the README gallery, plus a still.
- Every committed model/lottie must be self-made or CC0. Record provenance in `LICENSES.md` if CC0.
